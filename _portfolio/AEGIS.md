## Heading 
Worked on Level design for the Sea and Underworld sections of the game. Created a few models in game such as the broken chest and the Shield throw markers.
 This project was a submission for the Unlikely Collaborators Game jam. The theme of the game jam revolved around "Awe and Wonder and Change of perspective". I worked with 5 other people to develop the game over the course of 2 weeks. The work itself was pretty simple, I did run into some issues with the source control software as I had never used one with Unreal before but my teammates helped me out and I got the hang of it. The big difficulty was finding enough time to work on the game in the first place. I and all of my teammates were in the Senior Production class at the University, so the work for that asssignment took first priority. We had just gotten back from Spring Break so we were swamped with work that needed to be done. But with clear communication and scheduling, we managed to find the time to work on the game. At this moment the judging is still ongoing so I don't know if we won anything yet. I am hopeful that we did well, but regardless I am happy with the submission an I believe it is a solid game regardless.
