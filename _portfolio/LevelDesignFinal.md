## Heading 
Created the layout of the level using assets from Fab. The project was to design a "small world" in Unreal Engine. This project had slight difficulties as this was the only assignment that was built in the Unreal engine. The project was intended for use on an XR LED wall. Because of this the level needed to be wide enough to fit on the wall, 15 feet if I remember correctly, and needed to have depth. This meant movement in a close range, middle range, and far range. As this was the first and only project used in Unreal, the main difficulty was simply getting accustomed to the new engine. Fortunately I picked it up pretty quickly with the help of some YouTube tutorials. My biggest regret with this project is not getting to be able to see it on the XR wall, there was issues with the software and it did not get fixed before the year ended, but this assignment was a very enlightening experience to how game engines such as Unity and Unreal can be used outside of the game industry.
